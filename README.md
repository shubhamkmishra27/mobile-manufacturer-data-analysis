# 📱 Mobile Manufacturer Data Analysis using SQL

## 📌 Project Overview
This project focuses on analyzing mobile manufacturer data using SQL Server Management Studio (SSMS). The goal of this project is to extract meaningful business insights related to sales performance, customer trends, product analysis, and manufacturer performance using SQL queries.

- SQL Server Management Studio (SSMS)
- Data Analysis
- Relational Database Concepts

- KPI and aggregate analysis
- Business insight generation
